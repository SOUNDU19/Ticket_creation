## 🚀 Quick Start Guide

### 1. Setup (5 minutes)

**Windows:**
```bash
setup.bat
```

**Linux/Mac:**
```bash
chmod +x setup.sh
./setup.sh
```

### 2. Configure

Edit `backend/.env`:
```env
SECRET_KEY=your-secret-key
JWT_SECRET_KEY=your-jwt-secret
DATABASE_URL=sqlite:///nexora.db
```

### 3. Start Backend

```bash
cd backend
python app.py
```

Runs on: `http://localhost:5000`

### 4. Start Frontend

```bash
cd frontend
python -m http.server 8000
```

Runs on: `http://localhost:8000`

### 5. Login

**Default Admin:**
- Email: `admin@nexora.ai`
- Password: `admin123`

⚠️ Change password immediately!
