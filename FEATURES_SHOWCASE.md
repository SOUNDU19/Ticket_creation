## 🎯 Key Features Visualization

### AI Prediction Flow
```
User Input
    ↓
Text Preprocessing
    ↓
spaCy Lemmatization
    ↓
TF-IDF Vectorization
    ↓
ML Model Prediction
    ↓
Category + Priority + Confidence
    ↓
Entity Extraction
    ↓
Structured Output
```

### Authentication Flow
```
Login Form
    ↓
Validate Credentials
    ↓
Generate JWT Token
    ↓
Store in localStorage
    ↓
Protected Routes Access
```

### Ticket Creation Flow
```
Enter Description
    ↓
Click "Analyze with AI"
    ↓
API Call to /predict
    ↓
Display AI Results
    ↓
Review & Submit
    ↓
Create Ticket
    ↓
Redirect to History
```
