## Project Overview Questions

### Q1: Can you give me an overview of your NexoraAI project?
**Answer:** NexoraAI is an AI-powered customer support ticket automation system I built to streamline ticket management. It automatically categorizes support tickets, assigns priority levels, and provides intelligent routing using machine learning. The system features a modern SaaS-style interface with separate dashboards for users and administrators, real-time analytics, and achieves 94% accuracy in ticket classification.
